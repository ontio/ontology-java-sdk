## Introduction

This document provides a list of interfaces and their methods used in the Java SDK.

They are:

* Init 
* Communication with block chain interface
* Wallet manager 
* Digit asset
* Digit identity
* Neo smart contract deploy and invoke
* Native smart contract invoke
